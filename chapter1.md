# Section 4
## 1. 选择题
* 时间 + 地点 + 特征词（数字、地点、专有名词、引号、连字符、首字母大写）+ 主语
* 绝对选项排除(no,nothing,all...)
* 相似/相反选一个
* 谈到独立(independent)或预定(reservation)的选项应当选
* 钱的选项选最少的
* 除了地点和数字外其他选项都会改写

## 2. 答案句
* 最高级(most,best)
* 强调句(do)
* Actually, in fact, lastly, finally 后面
* 结论句(conclude, found that)
* used to do, rather than, instead of后面是被排除的

## 3. 地图题
* 路标 + 方位 + 答案
* 方位分为：东南西北和上下左右(top botton)

## 4. 搭配题
答案在定位词前后一句话
